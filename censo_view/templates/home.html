Bienvenidos a la practica sobre LinkedList y Arrays

En esta practica se implementa la Gestion de un Sistema de Censo, mediante el uso de archivos JSON. La finalidad de esta practica es la de comparar el rendimiento de las estructuras de datos LinkedList y Arrays.

Para ello se ha implementado un sistema de Censo, en el cual se pueden realizar las siguientes operaciones:

  • Registrar una Familia
  • Registrar un Generador para cada Familia
  • Eliminar una Familia y Generador
  • Actualizar los datos de Familias y Generadores
  • Ver todas las Familias registradas, Generadores, y el Censo con Familias que poseen un Generador
